# Hack University Initiative Worldwide

## Empowering Tomorrow's Innovators

Hack University Initiative Worldwide is a global ecosystem supported by student-led technology communities. The initiative connects students, builders, researchers, and industry mentors through hackathons, technical workshops, and collaborative projects.

Our mission is to make technology education, innovation, and entrepreneurship accessible to students from all backgrounds while fostering global collaboration.

This initiative is supported by the following organizations:

- Google Developer Group at Florida International University (GDG at FIU)
- Code Crunch
- ColorStack at FIU
- CAHSI at FIU
- Raspberry Pi Code Crunch

Together, these communities enable students to gain hands-on experience building real-world solutions, participating in hackathons, and connecting with industry professionals.
